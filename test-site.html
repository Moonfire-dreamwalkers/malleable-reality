Glassmorphism UI

This panel uses the .mr-panel class to achieve a glass-like blur effect over the ambient dark background. The glowing red text above uses .mr-text-accent.

Paragraphs with muted text use the .mr-text-muted class to establish hierarchy without needing inline styles.
